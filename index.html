❌ Solução de Problemas

Erros comuns e como resolvê-los

🔧 Problemas de Permissão

  • Erro: "Missing Permissions" - Verifique se o bot tem as permissões necessárias no servidor
  • Erro: "Cannot ban administrators" - Não é possível banir outros administradores
  • Erro: "Cannot execute command in DMs" - Alguns comandos só funcionam em servidores

🤖 Problemas do Bot

  • Bot não responde - Verifique se o bot está online e tem permissão de leitura no canal
  • Comandos não aparecem - Use /ajuda para recarregar a lista de comandos
  • Erro de rate limit - Aguarde alguns segundos antes de usar o comando novamente

📱 Problemas em Dispositivos Móveis

  • Botões não funcionam - Atualize o app do Discord para a versão mais recente
  • GIFs não carregam - Verifique sua conexão com a internet
  • Interface cortada - Use o modo paisagem para melhor visualização
